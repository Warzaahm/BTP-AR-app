# Application BTP simulation AR - Prototype

## Vision du projet
Application intuitive d'aide à la gestion de chantier. 
-Faciliter la prise en main d'un chantier
-Assurer la continuité d'un projet de chantier 
-Possibilité de scanner le chantier avec son appareil 
-Affichage par la réalité augmentée des tâches a effectuées
-Effectuer les contrôles et conformités
-Apllication intuitive et accessible par une l'intégration d'icône et système audio si possible

## Fonctionnalités
- Connexion ouvrier
- Scanner d'éléments de chantier
- Étapes de construction
- Contrôles de conformité

## Choix techniques
- **Frontend :** Svelte 5 + Vite, svelte-spa-router, @iconify/svelte, CSS
- **Simulation AR :** image de chantier en arrière plan annotée et un système d'overlay et hotspot positionnés avec coordonnées en pourcentage
- **Accessibilité :** Icônes 

## Installation
```bash

# Installer les dépendances
npm install

# Lancer en développement
npm run dev
```
